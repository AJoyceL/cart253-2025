## Art Jam: Self-Portrait

Joyce Angelina Lam


## Description
- This project is a stylised self-portrait, whose pupil motion is called by mouseX and mouseY.


## Contribution(s)
- The use of vertex() to draw the portrait.
- The use of map() to limit pupil mition range.
